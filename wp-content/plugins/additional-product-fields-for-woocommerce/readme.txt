== Description ==

https://www.youtube.com/watch?v=0Onykz2QK2A&t=1s

Extra Product Options Builder for WooCommerce lets you add more fields to your product page

= Features =
* **A bunch of fields at your disposition**
	* **Text Box**: Add one line of text ([Demo](http://productbuilder.rednao.com/product/text-box-demo/))
	* **Multiple Box**: Add multiple lines of text ([Demo](http://productbuilder.rednao.com/product/multiple-box-demo/))
	* **Radio:** Select from a list using radio buttons ([Demo](http://productbuilder.rednao.com/product/radio-demo/))
	* **Checkbox:** Select one or many options from a list ([Demo](http://productbuilder.rednao.com/product/checkbox-demo/))
	* **Paragraph:** Add additional text to the product ([Demo](http://productbuilder.rednao.com/product/paragraph-demo/))
	* **Divider:** Separate the extra product fields in sections ([Demo](http://productbuilder.rednao.com/product/divider-demo/))
	* **Datepicker:** Select a date from a calendar ([Demo](http://productbuilder.rednao.com/product/datepicker/))
	* **Masked Field:** Allow to input text in a predefined format ([Demo](http://productbuilder.rednao.com/product/masked-field))
	* **Color Picker:** Select a color form a dropdown ([Demo](http://productbuilder.rednao.com/product/color-picker/))
* **Conditional logic:** Show or hide using conditions, like when a variation is selected or when another field has an specific value  ([Demo](http://productbuilder.rednao.com/product/show-hide-demo/))

= Premium Features =

* **There is also a premium version that has more fields**
	* **List**: Add a list that can be increaded or decreased by the user ([Demo](http://productbuilder.rednao.com/product/list-demo/))
	* **Group Panel**: Create sub sections and display a sub total in each of them ([Demo](http://productbuilder.rednao.com/product/group-panel/))
	* **Repeter**: Create a list of fields that can be repeater with a buttons ([Demo](http://productbuilder.rednao.com/product/repeater/))
	* **File Upload**: Upload files ([Demo](http://productbuilder.rednao.com/product/file-demo/))
	* **Hidden**: Hiden field that is not visible by the user but can be seen in the order dashboard
	* **Image Picker**: Select one or many items from a list of images ([Demo](http://productbuilder.rednao.com/product/image-picker-demo/))
	* **Daterange**: Create a field that can do calculations between two dates ([Demo](http://productbuilder.rednao.com/product/daterange))
	* **Image Condition**: Change the product image depending on a condition. ([Demo](http://productbuilder.rednao.com/product/image-condition-demo/))
